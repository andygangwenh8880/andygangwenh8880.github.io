API Project

Context of Project: This project required students to choose an API that has information on a subject of their choice. Students are to provide the user with information on the subject using the information provided from the API.

Reflection: The API me and my group choose contained information on water consumption in New York City. To get the information we wanted to provide the user, we used a for loop to go through the API and push the information that we wanted to provide inside of an array. We were then able to print out the information on water consumption in New York City for the user by making a conditional that would grab the necessary information from the array after matching the year the information was about with the year the user had inputted. A challenge we encountered while working on this project was that this project restricted us from using HTML and CSS. Since we haven't learned many ways to add colors using JavaScript it was hard to make our project look more visually appealing to the user.
