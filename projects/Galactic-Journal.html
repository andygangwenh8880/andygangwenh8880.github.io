Galactic Journal

The Galactic Journal is created with HTML, CSS, and a JavaScript library known as Three.js. Three.js is a JavaScript library that helps a programmer create and display 3D graphics through WebGL(Web Graphics Library). WebGL is a JavaScript API that generates the 2D/3D graphics and animation that was programmed using Three.js onto compatiable web browsers. Programmers can use Three.js to code something simple like a rotating cube to something more complex such as a floating city.
